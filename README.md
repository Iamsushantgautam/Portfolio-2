---

## 🔥 Key Technical Highlights

* **Bespoke Animations:** Bypassed heavy generalized libraries by using `Framer Motion` to tie scale, rotation, and opacity transforms directly into scroll velocity data.
* **Component-Driven Design:** Built on a highly modular React 19 architecture using strict functional component separation (e.g., `HeroSection.jsx`, `ProjectsSlider.jsx`).
* **Fluid Page Dynamics:** Integrated `Lenis` (Studio Freight) to overwrite native scroll jitter with a smooth, frictionless navigation experience.
* **Aggressive Responsiveness:** Hand-coded robust media queries to perfectly format the layout from ultra-widescreen down to tight 320px mobile displays. No horizontal bleeding, no awkward overlapping.
* **Pop-Art Aesthetic:** Leveraged native CSS flex and grid matrices, layered with bold geometric boundaries, saturated CMYK accents, and high-contrast typography.

## 🛠️ Core Stack

* **Core:** React 19, JavaScript (ES6+), Vite 
* **Motion Engine:** Framer Motion
* **Scroll Hijacking:** Lenis
* **Styling Engine:** Raw, vanilla CSS with aggressive layout clamping
* **Icons Elements:** `lucide-react`, `react-icons`

## ⚙️ Quick Start Installation

You can pull the project down locally in seconds.

```bash
# 1. Clone the repository
git clone https://github.com/Iamsushantgautam/Portfolio.git

# 2. Scope strictly to the React build branch
git checkout react-portfolio

# 3. Install packages via NPM
npm install

# 4. Fire up the high-speed Vite development server
npm run dev
```
*The project will instantly spin up locally, usually on `http://localhost:5173`.*
